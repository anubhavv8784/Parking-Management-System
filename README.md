#  Parking Management System

A backend-based Parking Management System built using Node.js, Express.js, and MongoDB. It provides APIs to manage parking lots, parking slots, and vehicle parking operations.

##  Features

- Create and manage parking lots
- Manage parking slots
- Park and unpark vehicles
- Generate parking tickets
- MongoDB database integration
- RESTful APIs
- Environment variable support

##  Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv

##  Project Structure

```
src/
 ├── config/
 ├── controllers/
 ├── models/
 ├── routes/
 ├── services/
 └── server.js
```

##  Installation

```bash
npm install
```

##  Run Project

```bash
npm start
```

Development Mode:

```bash
npm run dev
```
